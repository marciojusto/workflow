## Acceptance Criteria

**AC4 – Editing Total Invoice Price recalculates discounts using delta excl. VAT**

Given the Summary is displayed
And a calculated Total Invoice Price is shown
When the dealer edits the Total Invoice Price manually
Then the system must:
1. Compute the delta between the original and the new Total Invoice Price (delta TTC)
2. Convert this delta to an excl. VAT value using: delta excl. VAT = delta incl. VAT / (1 + VAT%)
3. Add the delta excl. VAT to the "Other Discounts" field and update the discounts value on the summary section
4. Recalculate the new Total exc. VAT
5. Recalculate VAT amount and Total incl. VAT
6. Update all Summary fields accordingly
7. Display the new Total Invoice Price entered by the user

**Note 1:** The user should only reduce the amount.

---

## Implementation Summary

Added `originalTotalInvoicePrice` field to `AssetDealState` to preserve the first calculated Total Invoice Price. Modified `setPricingDetails` to initialize this value on first valid calculation.

Created `applyDeltaToOtherDiscount` async action in `assets.ts` that:
- Computes `deltaTTC = newValue - original`
- Converts to `deltaExclVAT = deltaTTC / (1 + VAT%)`
- Converts current Other Discount to euros (handling both € and % modes)
- Adds the absolute delta to Other Discounts
- Forces symbol to €
- Triggers `calculateDeal()` for backend recalculation

Modified `AssetSummary.vue` to make Total Invoice Price clickable (dashed underline). Clicking reveals a `q-input` for editing. On confirm:
- Validates that new value is not negative
- Validates that new value does not exceed original (shows Quasar `Notify` error if it does)
- Calls `applyDeltaToOtherDiscount` which updates Other Discounts and recalculates all Summary fields

Added `cannotIncreaseError` i18n keys for `en-US` and `fr-FR`.

Build passes successfully. E2E validation was attempted but could not proceed due to Renault SSO authentication requirements.

```diff
+    async applyDeltaToOtherDiscount(newTotalInvoicePrice: number) {
+      const original = this.originalTotalInvoicePrice ?? 0;
+      if (newTotalInvoicePrice > original) {
+        throw new Error("Total invoice price cannot exceed the original calculated value");
+      }
+      if (newTotalInvoicePrice < 0) {
+        throw new Error("Total invoice price cannot be negative");
+      }
+
+      const vatPercent = this.pricingDetails?.vatPercentage ?? 20;
+      const normalizedVat = vatPercent > 0 && vatPercent <= 1 ? vatPercent * 100 : vatPercent;
+      const deltaTTC = newTotalInvoicePrice - original;
+
+      if (Math.abs(deltaTTC) < 0.01) return;
+
+      const deltaExclVAT = deltaTTC / (1 + normalizedVat / 100);
+
+      const currentOther = Number(this.discount.otherDiscount || 0);
+      let currentOtherEuro = currentOther;
+      if (this.discount.otherSymbol === "%") {
+        const basePrice = Number(this.pricingDetails?.basePrice || 1);
+        currentOtherEuro = (currentOther / 100) * basePrice;
+      }
+
+      const newOtherEuro = currentOtherEuro + Math.abs(deltaExclVAT);
+
+      this.setOtherSymbol("€");
+      this.setOtherDiscount(Number(newOtherEuro.toFixed(2)));
+
+      await this.calculateDeal();
+    },
```
